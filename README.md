# NETRA Avatar Optimizer

Outil Unity Editor pour optimiser les avatars VRChat : physics bones & colliders, mesh, textures, blendshapes, materials.

## Installation via VCC

1. Ouvre Creator Companion
2. Va dans **Settings > Packages > Add Repository**
3. Colle l'URL : `https://YOUR_GITHUB_USERNAME.github.io/netra-vpm/index.json`
4. Dans ton projet, onglet **Manage Project**, trouve "NETRA Avatar Optimizer" et clique **+**

## Utilisation

Dans Unity : `Tools > NETRA Avatar Optimizer`
